Detailed Description

Represents a prepared statement.

Prepared statements will often (but not always) be faster than ordinary statements when reused several times.

They are used the same way as ordinary statements, except that they can be reset to their initial state and executed again. Their placeholder parameters are set by calling rebind(), which will reset the statement and then bind the provided parameters.
