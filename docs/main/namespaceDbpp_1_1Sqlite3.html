enum Dbpp::Sqlite3::OpenFlag : unsigned int
strong

Specifies how to open an SQLite3 database.

Since
v1.0.0
Enumerator
Memory 

The filename can be interpreted as a URI if this flag is set.

NoMutex 

The database will be opened as an in-memory database.

FullMutex 

The new database connection will use the "multi-thread" threading mode. This means that separate threads are allowed to use SQLite at the same time, as long as each thread is using a different database connection.

SharedCache 

The new database connection will use the "serialized" threading mode. This means the multiple threads can safely attempt to use the same database connection at the same time.

PrivateCache 

The database is opened with shared cache enabled, overriding the default shared cache setting.
